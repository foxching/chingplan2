
# CHINGPLAN

A simple project management tool with authentication and cloud functions in REACT.

Front End:

* React
* Redux
* React-Router
* Redux Thunk
* Materialize

 Backend
* Firebase and Firestore
* NodeJS 
* Express

and more

### Installation

Install all the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  

```sh
$ npm  start
```
